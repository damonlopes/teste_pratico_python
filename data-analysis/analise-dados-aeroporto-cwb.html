def translate_status(status):
    if status == "cancelled":
        return "Cancelado"
    elif status == "landed":
        return "Pousado"
    elif status == "scheduled":
        return "Agendado"
    elif status == "diverted":
        return "Desviado"
    elif status == "active":
        return "Em andamento"
    else:
        return "Desconhecido"

Concluses

De acordo com as informaes obtidas anteriormente, e com os valores e grficos gerados, podemos chegar a algumas concluses:

  • A maior parte do pblico que frequenta o aeroporto de Curitiba so pessoas que vieram ou esto se direcionando a outros aeroportos nacionais

  • A mdia de atraso no geral no acaba passando de 15 minutos, em que podem ocorrer alguns casos muito especiais de atraso

  • O aeroporto com maior nmero de conexes o aeroporto de Guarulhos, em So Paulo

  • So vrias companhias areas que passam pelo aeroporto de Curitiba, mas cerca de um tero so oriundas de companhias brasileiras
